Aplicatia Forum Volei Lovers
Dynamic Web Project -> JPA Facet

Prerequisites:

eclipse mars 2
mysql-installer-community-5.5.53.0
wildfly-9.0.2.Final
mysql-connector-java-5.1.33.jar
mysql-connector-java-5.1.33-bin.jar

Configurare Wildfly in consola pentru a folosi MySQL Server
MySQL JDBC Driver

Start MySQL Service inainte de a folosi aplicatia
Incarcare proiect in eclipse
Porneste server
Run as pentru aplicatie

Am folosit: JPA, Session beans, Servlets, JSP
